# Changelog

All notable changes to Rec0m88 will be documented in this file.

## [3.1.0] - 2026-02-13

### Added
- Recently Played Games Strip — Shows last 5 games with thumbnails on the idle screen
- Now Playing Status Bar — Shows game title, platform badge, and live play timer

## [3.0.0] - 2026-02-11

### Added
- Game Library Manager
- Automatic cover art and metadata fetching via RAWG & IGDB APIs
- Favorites system
- Game search functionality
- Fix Matches feature for incorrect metadata
- Delete Entry option (removes from DB, keeps ROM files)

## [2.1.0] - 2026-02-02

### Changed
- Massive netplay restructure for optimal performance

### Added
- Netplay messaging system

### Fixed
- Netplay guest audio issues
- Updated cheat files with working codes

## [2.0.0] - 2026-01-13

### Changed
- Moved to rec0m88.com domain

### Fixed
- Netplay stability (now lagless)
- Netplay audio synchronization
- Netplay video shift issues

## [1.1.1] - 2025-11-07

### Changed
- Cheat files are now user-editable
- Switched back to AppImage builds for Linux

### Fixed
- Save state naming inconsistency

## [1.1.0] - 2025-10-29

### Added
- Cheat code support
- Embedded cheats for NES, SNES, N64, Sega Genesis, and PlayStation

### Changed
- Backend emulator performance enhancements

## [1.0.4] - 2025-09-30

### Changed
- Netplay performance and stability enhancements

## [1.0.3] - 2025-09-12

### Changed
- Linux build changed from AppImage to .deb package

### Added
- Automatic update checking on startup
- Manual "Check for Updates" in Settings menu

## [1.0.2] - 2025-09-09

### Fixed
- Netplay host canvas centering
- 'About' window issue on macOS
- Alphabetical sorting of BIOS menu

## [1.0.1] - 2025-09-06

### Added
- Initial public release of Rec0m88
